<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

Make sure you have [Node.js](http://nodejs.org/) installed

### Installation

  ```sh
  $ git clone https://github.com/ANVRRT/Helper-19-Presentation-Website.git
  $ cd Helper-19-Presentation-Website
  $ npm install
  $ npm start
  ```
